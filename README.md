# Akshaya Charitable Trust Website

This folder contains only the static website files needed for GitHub Pages deployment.

Files included:
- `index.html`
- optimized WebP assets used by the homepage
- `.github/workflows/pages.yml` for GitHub Actions Pages deployment
- `.nojekyll`

To deploy:
1. Create a new GitHub repository.
2. Upload the contents of this folder to that repository.
3. Push to the `main` branch.
4. In GitHub repository settings, use `GitHub Actions` as the Pages source if prompted.
